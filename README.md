# formativeAssessment1
Description:<br/>
An app that provides suggestions for conversation based on the time of day inputted. This app allows you to maintain social relationships by providing you with a prompt as a starting point for a conversation.<br/>
Features: <br/>
The app contains a button, an input line to put in the time of day, and a suggestion that pops up after the input is considered.<br/>
Design Considerations:<br/>
The app is simplistic and easy to use in order to make it user friendly. It is straight to the point in order to help users start conversations as quickly as possible. <br/>

How to use: <br/>
Open the app <br/>

Check the time of day <br/>

Input the time of day where instructed <br/>

Click the "Click for suggestion" Button <br/>

Use the suggested output message to start a conversation! <br/>
